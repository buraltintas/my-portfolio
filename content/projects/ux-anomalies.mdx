## Highlight

Product-minded full-stack platform for documenting real-world UX problems.

## Overview

UX Anomalies is an Instagram-like platform where users can share unusual UX patterns, broken flows, and product oddities. Built with a Next.js frontend, Go backend, Cloud Run, Neon PostgreSQL, Google Cloud Storage, authentication, image uploads, likes, comments, public SEO pages, and sitemap support.
